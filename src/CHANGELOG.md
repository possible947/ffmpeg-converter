# Changelog
Все значимые изменения в проекте документируются в этом файле.  
Формат основан на принципах [Keep a Changelog](https://keepachangelog.com/).

---

## [1.1.0] — 2026‑01‑23
### Изменено
- Полная переработка архитектуры проекта.
- Модули (`audio`, `video`, `utils`, `progress`, `ffmpeg_cmd`, `core`) переведены на INTERFACE‑библиотеки.
- Реализация `.c` перенесена в `src/platform/<platform>/`.
- Добавлена полноценная поддержка Linux и macOS.
- Обновлён `src/cli/CMakeLists.txt` для платформенной сборки.
- Удалена зависимость от FFmpeg API — используется только внешний бинарник.
- Улучшена структура каталогов CLI.
- Добавлены README.md и таблица параметров CLI.

---

## [1.0.0] — 2025‑12‑??
### Добавлено
- Начальная версия проекта.
- Базовая логика формирования команд ffmpeg.
- Прогресс‑бар с FPS и ETA.
- Поддержка аудио‑нормализации (peak, loudnorm).
- Поддержка ProRes и профилей.
- Первичная версия CLI.
